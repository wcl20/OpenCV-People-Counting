# OpenCV People Counting
Count number of people moving left and right.


## Setup
Generate Virtual environment
```bash
python3 -m venv ./venv
```
Enter environment
```bash
source venv/bin/activate
```
Install required libraries
```bash
pip install opencv-python dlib imutils scipy 
```
Run program 
```bash
python3 main.py
```
